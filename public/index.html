Responsible Drinking Starts Here

QR-Based Alcohol Purchase Monitoring System using scientific tracking and daily limits

Key Features

QR Code Identity

Each user gets a unique QR code for secure identification at purchase points.

Scientific Tracking

Uses formula: Volume (ml) × Alcohol % × 0.789 to calculate pure alcohol grams.

20-Hour Limits

Automatic consumption reset after 20 hours with configurable limits for health safety.

Secure Authentication

JWT-based authentication with bcrypt password hashing and Aadhaar security.

Real-time Monitoring

Live tracking of consumption with instant approval/rejection decisions.

Complete History

Detailed transaction logs for users, vendors, and administrators.

About AlcoSecure

AlcoSecure is a responsible drinking initiative that uses technology to monitor and regulate alcohol consumption. By tracking pure alcohol intake using scientific formulas and enforcing daily limits, we promote healthier drinking habits.

Our system protects user privacy with secure Aadhaar hashing while providing accountability through QR-code based identity verification.
